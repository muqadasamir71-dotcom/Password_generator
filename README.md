# 🔐 Password Generator

> **Generate strong, secure, and random passwords with Python!**

---

## ✨ Features

- 🔢 Choose your own password length
- 🔠 Uppercase letters (A-Z)
- 🔡 Lowercase letters (a-z)
- 🔣 Numbers (0-9)
- 💥 Special symbols (!@#$%^&*)
- ⚡ Fast and easy to use
- 🛡️ Creates a different password every time

---

## 🛠️ Built With

- 🐍 Python 3

---

## 🚀 How to Run

1. Download or clone this repository.
2. Open `password_generator.py`.
3. Run the program.
4. Enter the password length.
5. Copy your secure password.

---

## 💻 Example

```text
===================================
       PASSWORD GENERATOR
===================================

Enter password length: 12

Generated Password:
A7#kP9@mQ2!x

===================================
```
